## Step-by-Step (30 Minutes)

### Step 1: Sign Up for Buffer (5 min)
```
1. Go to: https://buffer.com
2. Click "Sign Up"
3. Create account with email
4. Choose Pro plan ($15/month)
   - Unlimited posts
   - Best for: Auto-posting
```

### Step 2: Connect LinkedIn (3 min)
```
1. In Buffer dashboard, click "Add Social Account"
2. Select "LinkedIn"
3. Choose "Company Page" (not personal)
4. Authorize Buffer to access your page
5. Done!
```

### Step 3: Connect Facebook (3 min)
```
1. In Buffer dashboard, click "Add Social Account"
2. Select "Facebook Pages"
3. Choose your EZLY Facebook page
4. Authorize Buffer
5. Done!
```

### Step 4: Set Up Blog Feed (5 min)
```
1. In Buffer, go to "Integrations"
2. Find "RSS"
3. Enter your blog feed URL:
   https://ezly-dashboard.vercel.app/blog
   (or: https://ezly-dashboard.vercel.app/blog/feed.xml)
4. Click "Connect"
5. Choose which platforms: LinkedIn + Facebook
6. Done!
```

### Step 5: Test It (5 min)
```
1. Manually share your latest blog post:
   - Click "Create" in Buffer
   - Select one of your blog posts
   - Add custom message (or use template)
   - Choose LinkedIn + Facebook
   - Click "Schedule"

2. Buffer will:
   - Post to LinkedIn
   - Post to Facebook
   - Auto-optimize for each platform
```

### Step 6: Set Up Auto-Scheduling (4 min)
```
1. In Buffer, go to "Settings" > "Posting Schedule"
2. Set optimal times:
   - Monday 9 AM
   - Wednesday 2 PM
   - Friday 10 AM
3. Zones: Set to Mountain Time (MST)
4. Save!

Now Buffer will automatically post your blog
on these days/times to both platforms.
```

**Setup time:** 10 minutes

```

4. Action: LinkedIn - Create Share
5. Action: Facebook Page - Create Post
6. Done!

Cost: Free (up to 100 tasks/month)
Result: Auto-posts without Buffer
Downside: Less control, more manual tweaking
```

## Troubleshooting

**Posts not appearing?**
```
1. Check RSS feed is correct
2. Check Buffer shows new posts
3. Manually schedule one post to test
4. Check LinkedIn/Facebook pages
5. Wait 5-10 minutes for posts to show
```

**Posts look ugly?**
```
1. Edit post templates in Buffer
2. Add emojis, line breaks
3. Use brand hashtags: #EZLY #HomeImprovement
4. Test with manual post first
```

**Not getting engagement?**
```
1. Post more consistently (3x/week)
2. Use good images (already in blog posts)
3. Ask questions in caption
4. Reply to all comments
5. Share contractor success stories
6. Post on weekdays, avoid weekends
```
