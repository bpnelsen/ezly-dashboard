## Accessibility Considerations

### WCAG Compliance

#### AAA Level (Enhanced)
- Navy Blue (#0F3A7D) on White: ✅ 7.2:1 ratio
- Teal (#06B6D4) on White: ✅ 5.8:1 ratio
- Orange (#F97316) on White: ✅ 7.1:1 ratio
- All primary colors on white: ✅ WCAG AAA

#### AA Level (Minimum)
- All primary colors meet AA minimum (4.5:1 for normal text)
- All functional colors meet AA minimum

### Text Color Rules
```
On White Background:
- Use Navy Blue (#0F3A7D) for primary text
- Use Dark Gray (#1F2937) for secondary text
- Minimum font size 12px for secondary text

On Navy Blue Background:
- Use White (#FFFFFF) text (6.5:1 ratio)
- Use Light Gray (#F3F4F6) for secondary text

On Teal Background:
- Use White (#FFFFFF) text (5.8:1 ratio)
- Minimum 14px font size
```

## Color Usage Guide

### By Component Type

#### Buttons
```
Primary Button: Navy Blue background, white text
Secondary Button: Light Gray background, Navy Blue text
Tertiary Button: Transparent, Navy Blue text
Danger Button: Red background, white text
Success Button: Green background, white text
```

#### Forms & Inputs
```
Border: Light Gray (#E5E7EB)
Focus Border: Teal (#06B6D4)
Error Border: Red (#EF4444)
Success Border: Green (#10B981)
Background: White (#FFFFFF)
Label Text: Dark Gray (#1F2937)
Help Text: Medium Gray (#6B7280)
```

#### Cards
```
Background: White (#FFFFFF)
Border: Light Gray (#E5E7EB)
Heading: Navy Blue (#0F3A7D)
Body Text: Dark Gray (#1F2937)
Secondary Text: Medium Gray (#6B7280)
```

#### Navigation
```
Active Item: Navy Blue (#0F3A7D) with Teal underline
Inactive Item: Medium Gray (#6B7280)
Hover State: Light Gray background (#F3F4F6)
```

#### Status Badges
```
Accepted: Teal background (#06B6D4), white text
Pending: Amber background (#F59E0B), white text
Rejected: Red background (#EF4444), white text
Completed: Green background (#10B981), white text
```

## Color Standards

### Don'ts
❌ Create arbitrary color variations
❌ Mix different color systems
❌ Use colors without contrast checking
❌ Apply colors inconsistently
❌ Create new brand colors
❌ Use colors for decoration only

### Do's
✅ Use only approved colors
✅ Check accessibility ratios
✅ Maintain consistency across platforms
✅ Use colors with purpose
✅ Document any new usage patterns
✅ Update team when standards change
