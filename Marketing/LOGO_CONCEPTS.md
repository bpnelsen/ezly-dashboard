# EZLY Logo Concepts

**Purpose:** Reference guide for logo design options and specifications

---

## Logo Concept 1: Connected House Mark

### Concept Description
A modern geometric house shape with interconnected nodes representing the connection between homeowners and contractors.

### Design Elements
- **Primary Shape:** Minimalist house silhouette
- **Secondary Elements:** 3-4 connected circles/nodes
- **Connection Lines:** Subtle curves connecting the elements
- **Style:** Geometric, modern, clean

### Color Options
- Navy Blue (#0F3A7D) with Teal (#06B6D4) accents
- Single color Navy Blue for monochrome
- White knockout for dark backgrounds

### Usage
- Primary logo for all brand materials
- Scalable to 32px minimum
- Works well in square or horizontal formats

### Files Needed
```
logo-connected-house-horizontal.svg
logo-connected-house-stacked.svg
logo-connected-house-icon.svg
logo-connected-house-white.svg
```

A bold, geometric "E" that doubles as a visual representation of connection and building blocks.

## Logo Concept 3: Dual Symbol

### Concept Description
Represents both homeowner (house) and contractor (tools) united in one mark.

### Design Elements
- **Left Side:** Simplified house outline
- **Right Side:** Simplified tool outline
- **Center:** Connected/overlapping element showing unity
- **Style:** Minimalist, symmetric, balanced

### Color Options
- Navy Blue (#0F3A7D) house + Teal (#06B6D4) tools
- Single color variations
- White on dark

### Usage
- Primary logo for marketing materials
- Works horizontally and vertically
- Strong symbolic meaning for brand messaging

### Files Needed
```

logo-dual-symbol-vertical.svg

**PRIMARY RECOMMENDATION: Concept 1 + Concept 4 (Hybrid Approach)**

- **Concept 1 (Connected House)**: Excellent visual mark, scalable, meaningful symbolism

## Logo Specifications

### Primary Logo (Horizontal)
```
Format: SVG (vector)
Width: 200px minimum
Height: Auto (maintain aspect ratio)
Color: Navy Blue + Teal
File: logo-primary-horizontal.svg
```

### Logo Icon (Square)
```
Format: SVG + PNG (multiple sizes)
Sizes: 32px, 48px, 64px, 128px, 256px, 512px
Color: Navy Blue
Formats: SVG, PNG (with transparency)
Files: logo-icon-[size].svg, logo-icon-[size].png
```

```

Color: Navy Blue

## Clear Space & Sizing

### Minimum Clear Space
The clear space should be equal to the height of the main icon mark.

```
Clear Space (Horizontal): 20px on all sides
Clear Space (Vertical): 15px on all sides
Minimum Width: 200px (horizontal), 150px (vertical)
```

### Scaling Guidelines

```
DO NOT scale below:
- Icon: 32px
- Horizontal Logo: 200px width
- Vertical Logo: 150px width

DO scale proportionally:
- All logos use consistent aspect ratios
- Never distort or stretch
```

## DO'S & DON'TS

### DO ✅
- Use approved logo files
- Maintain clear space
- Scale proportionally
- Use color variations appropriately
- Place on sufficient contrast backgrounds
- Keep shape and proportions intact

### DON'T ❌
- Distort or stretch logo
- Add effects (shadows, gradients, 3D)
- Change colors arbitrarily
- Rotate or skew
- Use low-resolution versions
- Combine with competing imagery
- Violate clear space
